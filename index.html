Support Mapping Rules

      
    

    
REST = +Y upward arrow
HOLDDOWN = +/-Y double vertical arrows
GUIDE = lateral by pipe orientation: X pipe -> +/-Z; Z pipe -> +/-X; vertical pipe -> +/-X and +/-Z
LINE STOP / LIMIT / LIM = axial +/- pair unless explicit sign exists
SINGLE AXIS with missing +/- = warning marker / popupRequired
Can Spring / Spring Can = warning coil below pipe
Gap = record-scoped; no carry-forward
Axial restraints: no OD/2 radial contact; tips touch unless gap; positive gap = 10xgap separation
After engineering contact: visual resolver ODx2/3 applies only to final axial/pipe-parallel symbols
